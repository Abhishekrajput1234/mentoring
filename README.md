# Mentor Hub

Mentor Hub is a web application designed to connect students with experienced mentors for guidance in various fields.  
Users can browse mentors, book sessions, and communicate effectively to enhance learning and career growth.

## Live Demo
Live Demo : https://mentoring-alpha.vercel.app/ 

## Repository
GitHub Repository - 

## Features

- Browse and search for mentors by field or expertise
- View mentor profiles with experience and ratings
- Book mentoring sessions
- Contact mentors via built-in messaging
- User authentication and profile management
- Responsive and mobile-friendly UI

## Technologies Used

- **Frontend:** React.js, Tailwind CSS, Vite  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **HTTP Requests:** Axios  
- **Deployment:** Vercel (frontend), Render / Heroku (backend)
